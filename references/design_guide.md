# HTML PPT 视觉规范参考

> 本文档提供一组**示例性的视觉规范**。实际项目中，主色、点缀色、风格均可根据用户需求在 Markdown 文档中自由替换。

## 色彩变量（示例）

```css
:root {
  --bg:         #ffffff;      /* 背景色 */
  --surface:    #f8f9fa;      /* 卡片背景 */
  --accent:     #1a2a4a;      /* 主色 */
  --accent-2:   #c9a227;      /* 点缀色 */
  --text:       #2c3e50;      /* 正文 */
  --text-muted: #6c757d;      /* 次要文字 */
  --border:     #e2e8f0;      /* 边框 */
}
```

## 页面类型（示例）

| 类型 | 背景 | 文字 | 用法 |
|------|------|------|------|
| 深色封面/结束页 | 深色主色 | 白色 + 点缀色高亮 | 封面、抛出问题、结束页 |
| 过渡页 | 左半深色 + 右半浅色 | 左白右深 | 各部分开头 |
| 标准内容页 | `#ffffff` 或 `#f8f9fa` | `#2c3e50` | 正文、工具展示 |
| 引用/警示框 | `#f5f2e8` | `#2c3e50` | 底部强调、点缀色左边框 |
| 深色卡片 | 主色背景 | 白色 | 强调未来感/重点 |
| 代码框 | 深色背景 | 白色/点缀色 | 文件树、代码展示 |

## 字体规范

- 标题：Noto Sans SC / 思源黑体，主色
- 正文：深灰色，行距 1.5~1.8
- 点缀：点缀色用于关键词，避免大面积使用

## 动画

```css
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(18px); }
  to   { opacity: 1; transform: translateY(0); }
}
```

- `.anim-1` → 0s 延迟
- `.anim-2` → 0.1s 延迟
- `.anim-3` → 0.2s 延迟
- 只有 `.slide.active` 才触发

## 留白与边距

- 页边距：`1.5vw ~ 2.5vw`
- 内容最大宽度：`1600px ~ 1900px`
- 每页 3~5 个要点，不大段堆砌文字
