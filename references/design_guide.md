# HTML PPT 视觉规范参考

## 色彩变量
```css
:root {
  --bg:         #ffffff;      /* 背景色 */
  --surface:    #f8f9fa;      /* 卡片背景 */
  --accent:     #1a2a4a;      /* 主色：深海军蓝 */
  --accent-2:   #c9a227;      /* 点缀色：香槟金 */
  --text:       #2c3e50;      /* 正文 */
  --text-muted: #6c757d;      /* 次要文字 */
  --border:     #e2e8f0;      /* 边框 */
}
```

## 页面类型
| 类型 | 背景 | 文字 | 用法 |
|------|------|------|------|
| 深色封面/结束页 | `#0d1b2a` | 白色/香槟金 | 封面、抛出问题、结束页 |
| 过渡页 | 左半 `#0d1b2a` + 右半 `#ffffff` | 左白右深蓝 | 各部分开头 |
| 标准内容页 | `#ffffff` 或 `#f8f9fa` | `#2c3e50` | 正文、工具展示 |
| 引用/警示框 | `#f5f2e8` | `#2c3e50` | 底部强调、金边左边框 |
| 深色卡片 | `#1a2a4a` | 白色 | 强调未来感/Agent |
| 代码框 | `#0d1b2a` | 白色/金色 | 文件树、代码展示 |

## 字体规范
- 标题：Noto Sans SC / 思源黑体，深海军蓝
- 正文：深灰色，行距 1.5~1.8
- 点缀：香槟金用于关键词，不大面积使用

## 动画
```css
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(18px); }
  to   { opacity: 1; transform: translateY(0); }
}
```
- `.anim-1` → 0s 延迟
- `.anim-2` → 0.1s 延迟
- `.anim-3` → 0.2s 延迟
- 只有 `.slide.active` 才触发

## 留白与边距
- 页边距：`1.5vw ~ 2.5vw`
- 内容最大宽度：`1600px ~ 1900px`
- 每页 3~5 个要点，不大段堆砌文字
